CryptoTab是一个Google Chrome浏览器的扩展程序，目前只支持电脑版的Chrome，Chrome浏览器安装这个扩展后，只要打开浏览器，就可以自动挖比特币赚钱了。当前一个比特币的价格大概在一万美元下方波动。

<a href="#step2">第二：使用CryptoTab如何挣得更多</a>
<a href="#step3">第三：把挣得的比特币换成美元等法定货币</a>

<a name="step1"></a>
<h3>第一：CryptoTab安装使用图文教程</h3>
下面为大家详细介绍通过CryptoTab轻松赚钱的方法，首先打开Chrome浏览器，在Chrome浏览器中打开本页面，然后点击<a href="https://getcryptotab.com/412284" target="_blank">CryptoTab</a>打开<a href="https://getcryptotab.com/412284" target="_blank">CryptoTab网站</a>后，在网页中会看到一个“ADD TO CHROME AND GET? BITCOIN”的按钮（如下图），点击此按钮；
<img src="https://raw.githubusercontent.com/killgfw/fg/master/btc/cryptotab1.jpg" alt="CryptoTab 比特币赚钱" />

然后会弹出如下小窗口（从此窗口中可看到，这个CryptoTab扩展，当前在Google Chrome Web Store的用户评分接近五星级，参与评分的用户多达1万3千多人，总用户数当前有21万多，扩展是一个外国人发布在Google的 Chrome Web Store的，所以安全性应该是没问题的）；然后点击下图中的“添加扩展程序”按钮，这样就会把扩展安装到Chrome浏览器了；
<img src="https://raw.githubusercontent.com/killgfw/fg/master/btc/cryptotab2.jpg" alt="CryptoTab 比特币赚钱" />

安装后，在Chrome浏览器的右上角，会看到如下图的小按钮，点击这个按钮就可以打开CryptoTab界面：
<img src="https://raw.githubusercontent.com/killgfw/fg/master/btc/cryptotab3.jpg" alt="CryptoTab 比特币赚钱" />
点击如上图所示的，在Chrome浏览器的右上角的小按钮，就可以打开如下的界面：
<img src="https://raw.githubusercontent.com/killgfw/fg/master/btc/cryptotab4.jpg" alt="CryptoTab 比特币赚钱" />
上图中，标号1是当前挣得的比特币数量，标号2是你的推荐链接（图中是我的推荐链接：<a href="https://getcryptotab.com/412284" target="_blank">https://getcryptotab.com/412284</a>），把你自己的推荐链接发给朋友，发到自己的社交媒体或通过其它方式发给朋友，朋友注册后，你会得到比特币奖励。标号3的滑块是调节挖矿性能的，如果上网时感觉浏览器比较卡，可以把滑块调低一些；如果要离开电脑暂时不用浏览器，就把滑块调到MAX。标号4是一个菜单按钮，点击后会调出下图所示的界面：
<img src="https://raw.githubusercontent.com/killgfw/fg/master/btc/cryptotab5.jpg" alt="CryptoTab 比特币赚钱" />
在上图界面中的标号1、2、3分别是挣得的比特币总数，自己挖到的数量和介绍朋友挣到的数量。在这个界面往下拉，就出现如下界面：
<img src="https://raw.githubusercontent.com/killgfw/fg/master/btc/cryptotab8.jpg" alt="CryptoTab 比特币赚钱" />
在上图中，点击“LOG IN”（必须LOG IN，就是必须登录的意思，登录后才能保存挣到的比特币），点击后出现如下界面：
<img src="https://raw.githubusercontent.com/killgfw/fg/master/btc/cryptotab9.jpg" alt="CryptoTab 比特币赚钱" />
在上图中，可以选择用Facebook、Google或推特等账号来登录，根据自己的情况，点击Facebook、Google或推特按钮来登录，登录后在下图所示中，标号1处可以看到已经登录的信息（如果没有登录，则会显示一个蓝色的LOG IN的链接。）
<img src="https://raw.githubusercontent.com/killgfw/fg/master/btc/cryptotab10.jpg" alt="CryptoTab 比特币赚钱" />
上图中，标号2的复选框可以取消，不要显示那个钱币飘下来的动画。上图标号3，显示的是你介绍的朋友注册的数量。

再看下图：
<img src="https://raw.githubusercontent.com/killgfw/fg/master/btc/cryptotab7.jpg" alt="CryptoTab 比特币赚钱" />
在Chrome浏览器右上角，用鼠标右键点击CryptoTab的黄色小图标，则会弹出如上图的界面，我们可以选择Turn Off（关闭）挖矿，或选择挖矿所用的CPU占用量，分别是LOW（低）、Medium（中）、High（高），如果上网时感觉卡，则调低一点，等离开电脑或不用电脑时，就调为High（高）。但是，不要关闭电脑，也不要关闭Chrome浏览器，这样电脑就会自动持续挖矿为你赚钱了。

<h3>第三：把挣得的比特币换成美元等法定货币</h3>
首先，第一步，选择一个有法币（法定货币，比如美元、人民币等）交易的比特币交易所，注册一个账号，在交易所可以直接把比特币卖掉换成法币。

第二步，注册交易所后，在交易所里选择充币功能，会看到一个比特币地址。

第三步，在下图中，点击：“WITHDRAW BTC”,然后把第二步中注册的交易所中，在BTC（比特币）充币功能中看到的比特币地址，输入到这里，然后就把比特币转到交易所了。
<img src="https://raw.githubusercontent.com/killgfw/fg/master/btc/cryptotab5.jpg" alt="CryptoTab 比特币赚钱" />

最后一步，就是在交易所，利用法币交易功能，把比特币换成美元等法币，然后就可以转到自己的银行账户了。
